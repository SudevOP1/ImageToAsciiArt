# ImageToAsciiArt
Turn your images into cool ASCII art<br>
Just upload a photo and let the app convert it into a text-based version made of characters<br>
<br>

## üí° What it does
- Upload an image
- Convert it into ASCII art
- Shows the generated ASCII art
- You can copy the result
<br>

## üõ†Ô∏è Tech Stacks
- **Frontend**: React
- **Backend**: Django
<br>

## üöÄ How to run it locally

### 1. Clone the repo
```bash
git clone https://github.com/SudevOP1/ImageToAsciiArt.git
```
### 2. Backend Server
```powershell
cd ImageToAsciiArt/backend
pip install -r requirements.txt
python manage.py runserver
```
### 3. Frontend Server
```powershell
cd ImageToAsciiArt/frontend
npm install
npm run dev
```
### 4. See the magic happen
Go to `http://localhost:5173`
<br>
